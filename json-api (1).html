Documentation and Source for Javascript API

The Custom Search JSON API lets you develop websites and applications to retrieve and display search results from Programmable Search Engine programmatically.

RESTful requests can be done to get either websearch or img search in JSON format

Data format:

ATOM results format is no longer supported

The Custom Search JSON API uses the OpenSearch 1.1 Specification

Prereq:

You must first create a Programmable Search Engine. Please refer to the tutorial.html file to get it setup

Get JSON API Key:
https://developers.google.com/custom-search/v1/overview#:~:text=an API key.-,Get a Key,-Pricing

Pricing

Custom Search JSON API provides 100 search queries per day for free. If you need more, you may sign up for billing

If you need more than 10k queries per day and your Programmable Search Engine searches 10 sites or fewer, you may be interested in the Custom Search Site Restricted JSON API, which does not have a daily query limit

Monitoring:

Done via Cloud Platform Console’s API Dashboard

Intro:

JSON Custom Search API Overview:

There is only one method: list -> returns the requested search results -> GET

JSON API Data Model

The result of a search query to the Custom Search JSON API is a JSON object that includes 3 types of data:

  • Metadata describing the requested search
  • Metadata describing search engine
  • Search results

The Custom Search JSON API defines two custom properties and two custom query roles

Custom properties:

  • cx: The identifier of the Programmable Search Engine
  • safe: A description of the safe search level for filtering the returned results
    Custom query roles:
  • nextPage: A role that indicates the query can be used to access the next logical page of results
  • previousPage: A role that indicates the query can be used to access the previous logical page of results, if any.

Using REST:

somewhat different from traditional REST

the API provides access to a service. As a result, the API provides a single URI that acts as the service endpoint.

You can retrieve results for a particular search by sending an HTTP GET req

format for the Custom Search JSON API URI is: https://www.googleapis.com/customsearch/v1?[parameters]
Three query [parameters] are required with each request:

  • API Key - use the key query param to identify your application
  • Prog Search Eng ID - use cs to specify the Programmable Search Engine you want to use to perform this search.
